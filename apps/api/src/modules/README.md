# Domain modules

NestJS domain modules for the modular monolith.

## Implemented

- **identity** — authentication, JWT sessions, roles, permissions, audit foundation
- **tenancy** — tenants, organizations, memberships, tenant context, tenant switching
- **providers** — provider profiles, qualifications/certifications/languages, weekly availability, verification submit/review, public profiles
- **services** — categories, service catalog CRUD, draft/publish/pause/archive workflow, media/tags/locations/FAQs/requirements, storage port abstraction
